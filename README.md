### 本项目简介
本项目主要用于快速搭建es6+react的开发环境，配置步骤可以参考[笔记](http://note.youdao.com/noteshare?id=85e8d4e047c5e198f0b09274c1003cd5),后续会在上面写一些demo，供大家学习和参考。
